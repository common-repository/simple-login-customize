== Description ==

Simply customize your admin login page using this plugin. 

A few notes about the sections above:

*   Customize admin login logo
*   Change login with username only.
*   Hide lost password reset link
*   Hide back to site link
*   Change/Custom error message
*   Dynamic image loading with your choice

Also see my other plugins 
* [Easy Subscribe Button Widget](https://wordpress.org/plugins/widget-youtube-subscribtion/ "Easy Subscribe Button Widget") <br>
* [Easy Embed Page Widget](https://wordpress.org/plugins/embed-page-facebook/ "Easy Embed Page Widget")

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/simple-login-customize` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
3. Configure it (the plugin doesn\'t do anything until configured)
4. Verify that it works
